# CloudOps Mentor AI

CloudOps Mentor AI is a memory-powered DevOps assistant built for Quackathon 2026.

## Features

- Persistent memory using Parcle
- Cloud troubleshooting assistant
- Memory retrieval for previous incidents
- Flask backend API
- Simple chat interface
- Memory Vault & Analytics window to see the recent memory stored & Dashboard.

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Flask (Python)
- Memory Layer: Parcle AI

## Running Locally

1. Install dependencies

pip install -r requirements.txt

2. Create .env

PARCLE_API_KEY=your_key

3. Run

python app.py

4. Open

http://localhost:8000
